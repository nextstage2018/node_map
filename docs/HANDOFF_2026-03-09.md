### バグ1: マイルストーン展開時にエラー画面

**現象**: タスクタブでマイルストーンをクリック（展開）するとエラー画面になる

**原因**: `src/components/v2/MilestoneSection.tsx` 124行目
```tsx
// MilestoneCard は MilestoneSection とは別の関数コンポーネント
// projectId が MilestoneCard のスコープに存在しない
<MilestoneEvaluation
  milestoneId={milestone.id}
  milestoneTitle={milestone.title}
  projectId={projectId}  // ← ここで undefined になる
  onStatusUpdate={handleStatusUpdate}
/>
```

**修正方法**: `MilestoneCard` に `projectId` を props で渡す
```tsx
// MilestoneCard の props に projectId を追加
function MilestoneCard({ milestone, projectId, onStatusUpdate }: {
  milestone: Milestone;
  projectId: string;
  onStatusUpdate: (id: string, status: string) => void;
}) {

// 呼び出し側（MilestoneSection内）
<MilestoneCard
  key={ms.id}
  milestone={ms}
  projectId={projectId}  // ← 追加
  onStatusUpdate={handleStatusUpdate}
/>
```

**ファイル**: `src/components/v2/MilestoneSection.tsx`
